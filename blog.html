Question-1: What are the different ways to select on element in the DOM?

Answer: There are multiple ways to select an element in the DOM. Some of the most common ways are:

  • document.getElementById()
  • document.getElementsByClassName()
  • document.getElementsByTagName()
  • document.querySelector()
  • document.querySelectorAll()

Question-2: What is the difference between InnerHTML, InnerText and TextContent?

Answer: The difference between InnerHTML, InnerText and TextContent is as follows:

  • InnerHTML: It returns the HTML content of an element.
  • InnerText: It returns the text content of an element.
  • TextContent: It returns the text content of an element and all its descendants.

Question-3: What is event delegation in the DOM?

Answer: Event delegation is a technique in which we attach a single event listener to a parent element, instead of attaching multiple event listeners to multiple child elements. This is useful when we have a large number of child elements and we want to handle events on them.

Question-4: What is event bubbling in the DOM?

Answer: Event bubbling is a mechanism in which an event is first captured by the innermost element and then propagated to the outer elements. This means that when an event occurs on an element, it will first trigger the event handlers on that element, then on its parent element, and so on, until it reaches the root element.

Question-5: How do you create, add, and remove elements using JavaScript?

Answer: To create, add, and remove elements using JavaScript, you can use the following methods:

  • document.createElement(): To create a new element.
  • parentElement.appendChild(): To add a new element to the DOM.
  • parentElement.removeChild(): To remove an element from the DOM.
